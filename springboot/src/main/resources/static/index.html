Shadownet Nexus API

Spring Boot backend for Solo, Missions, Story, and the shared narrator event system.

Runtime

  • Frontend: http://localhost:8080 by default; Vite may choose the next port if 8080 is busy.
  • Backend: http://localhost:3001
  • Health: /actuator/health
  • Database: MySQL on Docker port 3305 or native port 3306.

Player Routes

Solo: ranked CTF with non-ranked narrator training after repeated misses.
Missions: trust-driven operations with mission runtime cell mechanics embedded.
Story: operator narrative at /story/operator/:id.

API Surface

POST /api/auth/register
POST /api/auth/login

GET  /api/operators
GET  /api/challenges
GET  /api/missions
GET  /api/story/progress

GET  /api/puzzle/session/{challengeId}
POST /api/puzzle/submit

POST /api/team/create
GET  /api/team/{teamId}

Security

JWT authentication
Rate limiting
Audit logs
Backend-authoritative score, mission, and story state
